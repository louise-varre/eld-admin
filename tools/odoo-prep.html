How this works
  • Screenshots: drop Shopify order screenshots, Claude reads them and produces the v3 block.
  • CSV: paste a Shopify CSV export instead.
  • Manual DDP: the old spreadsheet, but live. Edit margin rates per country if they ever change. Generate block reads from these inputs, no API call needed.
  • The Checklist tab on the right shows a step-by-step list — click each value to copy, tick the box once it's typed into Odoo. Progress bar tracks completion.
  • Raw block tab is still available for the full text version if you prefer copying that.
  • Workflow: keep this window next to Odoo. For each step in the checklist, click the value to copy, paste into Odoo, tick the checkbox.
